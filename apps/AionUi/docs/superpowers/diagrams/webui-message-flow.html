端到端消息流转（以"发送 AI 对话"为例）

步骤 位置 关键调用
① 用户输入 React 组件 (renderer) bridge.emit('task:send', { message })
② 浏览器适配 src/common/adapter/browser.ts socket.send(JSON.stringify({ name, data }))
③ WS 接收 WebSocketManager.onmessage JSON.parse → onMessage(name, data, ws)
④ 注入 bridge src/process/webserver/adapter.ts getBridgeEmitter().emit('task:send', data)
⑤ 路由到 handler src/common/adapter/main.ts bridge.on ipcMain emitter.emit → taskBridge.provider
⑥ 任务调度 WorkerTaskManager AgentFactory.create() → worker.fork()
⑦ AI 流式输出 Worker 子进程 process.send({ chunk }) → IpcAgentEventEmitter
⑧ 推送 bridge taskBridge / IpcAgentEventEmitter bridge.emit('task:chunk', { chunk })
⑨ 广播出口 src/common/adapter/main.ts bridge.adapter.emit wsManager.broadcast + win.webContents.send
⑩ 浏览器接收 src/common/adapter/browser.ts socket.onmessage emitter.emit('task:chunk', data) → React 组件更新

WebUI 认证流程（二维码登录）

传统账号密码
1POST /auth/login { username, password }
2AuthService.verifyPassword (bcrypt)
3返回 JWT sessionToken
4后续请求 Header: Authorization: Bearer <token>
二维码扫码登录
1Electron 端生成 32 字节随机 QR Token (5min TTL)
2构建 http://<host>:<port>/qr-login?token=xxx
3手机扫码 → GET /qr-login?token=xxx
4verifyQRTokenDirect → 验证 TTL + 本地 IP 限制 + 一次性
5AuthService.generateToken → 返回 JWT sessionToken
